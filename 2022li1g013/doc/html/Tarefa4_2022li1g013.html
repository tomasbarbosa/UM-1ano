jogoTerminou #

Arguments

:: Jogo

recebe o jogo

-> Bool

devolve um boolean

A função jogoTerminou é usada para verificar se o Jogo terminou devido ao Jogador estar fora do Mapa, ter caído na Água ou estar debaixo de um Carro

A função pode ser definida da seguinte forma : 

jogoTerminou (Jogo (Jogador (x,y)) (Mapa l terr)) 
    | x || x= l || y || y= length terr = True 
    | otherwise = caiunaagua (x,y) (terr !! y) || debaixodecarro (x,y) (terr !! y)

A função caiunaagua é usada para verificar se o Jogador caiu na Água

A função pode ser definida da seguinte forma : 

caiunaagua (x,y) (Rio v, obstaculos)
    | obstaculos !! x == Nenhum = True
    | otherwise = False
caiunaagua _ _ = False
