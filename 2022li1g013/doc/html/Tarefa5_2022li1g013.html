-> Jogo

devolve o jogo atualizado com a linha nova

A função deslizaJogo é usada para adicionar uma linha ao topo do Mapa e retirar a última, ajustando as Coordenadas do Jogador

A função pode ser definida da seguinte forma : 

deslizaJogo (Jogo (Jogador (x,y)) (Mapa l terr)) n  = (Jogo (Jogador (x,y+1)) (removeultimalinha (estendeMapa (Mapa l terr) n)))

A função removeultimalinha é usada para retirar a última linha do Mapa

A função pode ser definida da seguinte forma : 

removeultimalinha (Mapa l terr) = (Mapa l (take (length terr - 1) terr))
